Om Bulma:

Bulma[1] är ett Open-Source CSS Ramverk som erbjuder klara frontend-komponenter som enkelt kan kombineras och användes för att bygga responsive moderna webbplatser. Personligen så gillar jag de standard komponenter som ramverket erbjuder såsom menyn, navigationen o.s.v. Men är inget större fan utav hur dom har strukturerat meny hiden, själva menyn gömmer sig själv på mindre enheter för att man ska använda burger "elementet" för dessa enheter.

vilket i sig själv är resonabelt, men burger elementet kräver javascript för att fungera i huvud taget, vilket lät onödigt jobbigt så jag valde att tvinga menyn att visa sig på mobila enheter istället, fast då har den helt bytt format och blir enkelt sagt, skitful. Det finns kanske inget bättre sett att strukturera på men är inte en entusiast för denna "approach"

Pluspoäng dock för enkelheten i att ändra paddings, margins på de olika klasserna utan att behöva modifiera css'en, skönt när man vill göra små ändringar på element utan att behöva måtta för mycket.
