Om Foundation:

Foundation är ett responsivt design ramverk som inehåller både css och javascript. Tyckte om det en del då det är relativt lätt vikt och inte tränger sig in så mycket på den generella designen. Den var mycket enklare än Bulma att jobba runt fast mindre kraftfull på CSS delen, mest användbar för responsiva funktioner såsom grids och formulär/tabeller. Däremot så finns det en hel databas på deras webbplats med olika html/css/javascript snippets som låter en i stort sett bygga sidan med minimal egen kod, dessa kanske är bra eller dåliga. Vet helt ärligt inte då jag valde att inte använda dem till denna sida.

I min åsikt så ser Bulma mycket bättre ut, fast det är ett rent CSS ramverk, medans det här är mer praktiskt (i min åsikt) och skapar inte html kod som ser ut som ett arkiv (div/div/div/div). Rent funktionellt så kan detta vära "snäppet värre" men det är svårt att forma en riktig åsikt om det utan erfarenheten som krävs för att göra det bedömmandet.
